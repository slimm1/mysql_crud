========================
 MYSQL CRUD CON mariadb
========================

>> El archivo jar se encuentra en la carpeta dist.

>> El programa simula una base de datos con usuarios y shows asociados a cada usuario. El programa funciona con una 
base de datos mariadb en el puerto 3306. asegurate de tener el servicio mysql iniciado antes de ejecutarlo.

>> Para no complicarme, un usuario solo puede tener un show asociado.

>> Eliminar usuario: Selecciona un item en la tabla y pulsa el botón de eliminar.

>> Añadir usuario: Introduce los datos que quieras y pulsa el botón de añadir.

>> Editar usuario: Selecciona un item de la tabla y cambia algo antes de pulsar editar.

>> El menú desplegable con opciones en la venta usuarios representa los shows disponibles en ese momento para 
asignar a un determinado usuario

>> CLEAR: limpia la selección de la tabla y los campos.

>> Shows: Abre la ventana para editar, añadir o eliminar shows.
